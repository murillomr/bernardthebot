# bernardthebot

BOT para buscar a programação de canais favoritos. 

Para adicionar um novo canal, é necessário:

- Alterar o request no arquivo TV_montaURL.py, de acordo com o ID do canal procurado.
- Alterar o arquivo TV.py com uma nova função para tratar o ID do canal procurado.
- Alterar o arquivo bernard.py com uma nova função para envio da mensagem e adicionar a chama da função do fim do arquivo.
